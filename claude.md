# AI Guidance File

## Coding Principles

- Always validate user inputs before database operations
- Keep business logic separate from route handlers
- Use clear and readable code over clever code
- Prevent invalid states using constraints
- Handle errors gracefully

## Architecture Rules

- Flask handles API layer
- SQLAlchemy handles database ORM
- React handles frontend UI
- Frontend communicates only through API

## AI Usage

AI was used to:

- Scaffold initial project structure
- Generate boilerplate code
- Suggest validation and improvements

All generated code was reviewed and verified before use.
